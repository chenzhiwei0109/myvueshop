# myvueshop

## 文件引入
```
npm install || cnpm install
```

### 项目启动
```
npm run serve
```

### 后端启动
项目需要使用kv数据库mongodb
```
cd shop_server
npm install  || cnpm install
node index.js
```

### 加载数据库
```
地址栏输入
http://localhost:3000/product/insertProductInfo //加载产品
http://localhost:3000/type/insertType  //加载分类

node index.js  //重启服务
```
